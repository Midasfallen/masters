# Тестовые изображения для MinIO

Скрипт `npm run upload-test-images` загружает изображения из двух подпапок в MinIO:

| Папка    | Назначение              | Кол-во | Имена в MinIO        | Бакет  |
|----------|-------------------------|--------|----------------------|--------|
| `posts/` | Медиа постов в ленте    | 24     | test-1.jpg … test-24.jpg | posts  |
| `avatars/` | Аватары пользователей | 11     | avatar-1.jpg … avatar-11.jpg | avatars |

Положите сюда JPG или PNG. После загрузки выполните:

```bash
npm run upload-test-images
```

Затем перезапустите сид (если нужно): `npm run seed`.
