# BRAND BOOK - Service Platform

**Версия:** 1.0
**Дата:** Декабрь 2025

---

## 1. Позиционирование бренда

### Миссия
Связать людей с профессионалами в сфере услуг через визуальный, современный и удобный опыт.

### Видение
Стать главной платформой для поиска и бронирования услуг beauty, здоровья, спорта и образования через визуальный контент.

### Ценности
- **Визуальность** - показываем, а не рассказываем
- **Доверие** - прозрачные отзывы и верификация
- **Удобство** - 3 тапа до записи
- **Качество** - фокус на профессионалах
- **Сообщество** - социальные функции и вовлечённость

### Tone of Voice
- **Дружелюбный** - тёплое общение, без формализма
- **Современный** - актуальные термины, молодёжный язык
- **Профессиональный** - экспертность в услугах
- **Вдохновляющий** - мотивируем заботиться о себе

## 2. Визуальная идентичность

### 2.1 Логотип

**Концепция:**
Минималистичный символ, сочетающий буквы "S" (Service) с элементами:
- Звезда качества (рейтинг)
- Точка геолокации (поиск рядом)
- Человек (персональный сервис)

**Варианты:**

```
┌─────────────────────────────────────────────────┐
│  Logo Horizontal                                │
│                                                 │
│   ⭐                                            │
│  ╱ ╲   SERVICE                                 │
│ ╱   ╲  Platform                                │
│ ╲   ╱                                          │
│  ╲ ╱                                           │
│   ▼                                             │
│                                                 │
└─────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────┐
│  Logo Icon Only (App Icon)                     │
│                                                 │
│     ╔═══╗                                      │
│     ║ S ║  ← Минималистичная "S"              │
│     ╚═══╝     с градиентом                     │
│                                                 │
└─────────────────────────────────────────────────┘
```

**Типографика логотипа:**
- **SERVICE** - Inter Bold, 24pt
- **Platform** - Inter Regular, 14pt

**Охранное поле:**
Минимальное пространство вокруг логотипа = высота символа "S"

**Минимальный размер:**
- Digital: 24px высотой
- Print: 15mm высотой

**Недопустимо:**
- ❌ Искажение пропорций
- ❌ Изменение цветов без согласования
- ❌ Добавление эффектов (тень, обводка)
- ❌ Поворот логотипа

### 2.5 Imagery (Изображения)

#### **Стиль фотографий**

**Основные принципы:**
- ✅ Естественное освещение
- ✅ Реальные люди, реальные услуги
- ✅ Фокус на результате работы мастера
- ✅ Тёплые, приветливые тона
- ✅ Высокое качество (минимум 1080p)

**Недопустимо:**
- ❌ Стоковые "пластиковые" фото
- ❌ Слишком постановочные кадры
- ❌ Перенасыщенные фильтры
- ❌ Размытые или низкокачественные фото

#### **Соотношения сторон**

```
Портфолио мастера:    1:1 (квадрат)
Обложка профиля:      16:9 (широкий)
Посты в ленте:        4:5 (вертикальный)
Stories:              9:16 (полноэкранный)
Аватары:              1:1 (круг)
```

#### **Обработка**

**Фильтры (опционально):**
- Лёгкое повышение контраста (+10%)
- Тёплый баланс белого (+5% warm)
- Лёгкая виньетка по краям
- БЕЗ heavy фильтров (Instagram-style)

#### **Placeholders**

Для отсутствующих фото использовать:
- Градиент Primary → Secondary
- Иконка категории в центре (белая, 40% opacity)

**Material 3 тени:**
```css
/* Level 1 */
```

## 4. Анимации и переходы

### 4.1 Timing Functions

```
Standard:    cubic-bezier(0.4, 0.0, 0.2, 1)  - общие переходы
Decelerate:  cubic-bezier(0.0, 0.0, 0.2, 1)  - входящие элементы
Accelerate:  cubic-bezier(0.4, 0.0, 1, 1)    - уходящие элементы
Sharp:       cubic-bezier(0.4, 0.0, 0.6, 1)  - быстрые переходы
```

### 4.2 Длительности

```
Extra Short:  50ms   - мгновенные feedback
Short:        100ms  - hover эффекты
Medium:       200ms  - стандартные переходы
Long:         300ms  - сложные анимации
Extra Long:   500ms  - page transitions
```

### 4.3 Типы анимаций

#### **Fade (Затухание)**
```dart
duration: 200ms
curve: standard
opacity: 0.0 → 1.0
```

#### **Scale (Масштабирование)**
```dart
duration: 300ms
curve: decelerate
scale: 0.8 → 1.0
Использование: Dialog появление
```

#### **Slide (Сдвиг)**
```dart
duration: 300ms
curve: decelerate
offset: (0, 50) → (0, 0)
Использование: Bottom sheet
```

#### **Shared Element Transition**
```dart
duration: 300ms
curve: standard
Использование: Карточка → Детальная страница
```

## 6. Accessibility (Доступность)

### 6.1 Контрастность

**WCAG AAA Standard:**
- Обычный текст: минимум 7:1
- Крупный текст (18pt+): минимум 4.5:1
- UI элементы: минимум 3:1

**Проверка палитры:**
```
Primary (#6750A4) на White (#FFFFFF):     ✅ 7.2:1
Body text на Surface:                     ✅ 8.5:1
Label на Primary Container:               ✅ 4.8:1
```

### 6.2 Touch Targets

**Минимальные размеры:**
- Buttons: 48dp × 48dp
- Icons: 48dp × 48dp (включая padding)
- List items: минимум 48dp высотой
- Spacing между targets: минимум 8dp

### 6.3 Screen Reader

- Все кнопки и ссылки: описательные labels
- Изображения: alt text
- Формы: связанные labels
- Динамический контент: announcements

## 9. Чеклист соответствия

При создании нового UI проверьте:

- [ ] Используются цвета из палитры брендбука
- [ ] Типографика соответствует спецификации
- [ ] Радиусы скругления из системы (4dp, 8dp, 12dp, 16dp)
- [ ] Spacing кратен 4dp
- [ ] Минимальный touch target 48dp × 48dp
- [ ] Контрастность текста минимум 7:1
- [ ] Используются Material 3 компоненты
- [ ] Анимации 200-300ms с правильными easing
- [ ] Иконки из Material Symbols Rounded
- [ ] Тон общения дружелюбный и на "ты"
