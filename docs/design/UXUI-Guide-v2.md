## 1. User Flow

### 1.1 Общий флоу приложения (v2.0)

```
Запуск приложения
    │
    ├─→ Первый запуск?
    │   ├─ ДА → Onboarding (3 экрана)
    │   │         │
    │   │         └─→ Регистрация (Email, Телефон, Имя, Пароль)
    │   │               │
    │   │               ├─→ Запрос разрешения геолокации
    │   │               │
    │   │               └─→ Feed (лента контента)
    │   │
    │   └─ НЕТ → Проверка авторизации
    │               ├─ Авторизован → Feed
    │               └─ Не авторизован → Экран входа
    │
    └─→ Feed (Главная лента)
          │
          ├─→ Bottom Navigation (5 вкладок)
          │   ├─ 🏠 Feed (Лента контента)
          │   ├─ 🔍 Поиск (Поиск + каталог)
          │   ├─ 💬 Чаты (Список чатов)
          │   ├─ 📅 Записи (Клиент/Мастер/Все)
          │   └─ 👤 Профиль (+ уведомления)
          │
          └─→ Действия пользователя
              ├─ Просмотр постов (полноэкранно)
              ├─ Поиск мастера
              ├─ Просмотр профиля мастера
              ├─ Подписка на мастера
              ├─ Добавление в друзья
              ├─ Создание записи
              ├─ Чат с мастером/друзьями
              ├─ Лайк/комментарий/репост
              ├─ Оставить отзыв (обязательно!)
              └─ Стать мастером (опционально)
```

### 1.2 Флоу клиента (v2.0): Запись к мастеру через Feed

```
1. Feed (главная страница)
   │
   ├─ Лента постов (3 столбца)
   │  │
   │  └─→ 2. Клик на пост
   │        │
   │        └─→ 3. Полноэкранный просмотр
   │              ├─ Просмотр фото/видео (свайп между медиа)
   │              ├─ Лайк ❤️
   │              ├─ Комментарий 💬
   │              ├─ Поделиться 📤
   │              ├─ В избранное ⭐
   │              │
   │              └─→ 4. Кнопка "Записаться" 📅
   │                    │
   │                    └─→ 5. Переход в профиль мастера
   │                          ├─ Портфолио
   │                          ├─ Услуги и цены (развернуть)
   │                          ├─ Отзывы
   │                          └─ Выбор услуги
   │                             │
   │                             └─→ 6. Выбор даты и времени
   │                                   ├─ Календарь с доступными слотами
   │                                   └─ Выбор конкретного времени
   │                                      │
   │                                      └─→ 7. Подтверждение записи
   │                                            ├─ Детали: Услуга, Дата, Время, Цена
   │                                            ├─ Комментарий (опционально)
   │                                            └─ [Создать запись]
   │                                               │
   │                                               └─→ 8. Запись создана ✅
   │                                                     ├─ Статус: "Ожидает подтверждения"
   │                                                     ├─ Уведомление мастеру
   │                                                     ├─ Чат с мастером доступен
   │                                                     │
   │                                                     └─→ 9. Мастер подтверждает
   │                                                           ├─ Push уведомление клиенту
   │                                                           ├─ Напоминание за 1 час
   │                                                           │
   │                                                           └─→ 10. После визита
   │                                                                 ├─ Мастер завершает (с отзывом)
   │                                                                 └─ Клиент ОБЯЗАН оставить отзыв
```

### 1.3 Флоу клиента: Альтернативный путь через Поиск

```
1. Поиск (2-я вкладка)
   │
   ├─ Текстовый поиск (имя мастера/услуга)
   │  │
   │  └─→ Результаты по табам:
   │        ├─ Мастера
   │        ├─ Услуги
   │        ├─ Посты
   │        └─ Пользователи
   │
   ├─ Многоуровневый каталог
   │  │
   │  └─→ Выбор категории (например "Красота")
   │        │
   │        └─→ Подкатегория (например "Маникюр")
   │              │
   │              └─→ Результаты по табам:
   │                    ├─ Мастера (список по геолокации)
   │                    └─ Посты (сетка 3 столбца)
   │
   └─ Фильтры
      ├─ Геолокация (1-50 км, вся страна)
      ├─ Категория
      └─ Цена
         │
         └─→ Переход к профилю мастера → (см. флоу выше, шаг 5)
```

### 1.4 Флоу мастера: Создание профиля

```
1. Профиль (5-я вкладка)
   │
   ├─ Кнопка "Стать мастером" ⭐
   │  │
   │  └─→ 2. Flow создания профиля мастера (5 шагов)
   │        │
   │        ├─ Шаг 1/5: Основная информация
   │        │    ├─ Имя мастера
   │        │    └─ Био (до 500 символов)
   │        │
   │        ├─ Шаг 2/5: Выбор категорий
   │        │    └─ Выбрать из каталога (1-5 категорий)
   │        │
   │        ├─ Шаг 3/5: Услуги и цены
   │        │    ├─ Название услуги
   │        │    ├─ Категория
   │        │    ├─ Цена
   │        │    ├─ Продолжительность
   │        │    └─ Описание + медиа
   │        │
   │        ├─ Шаг 4/5: Расписание
   │        │    ├─ Рабочие дни
   │        │    ├─ Часы работы
   │        │    └─ Перерывы
   │        │
   │        └─ Шаг 5/5: Локация
   │             ├─ Базовый адрес
   │             ├─ Радиус выезда
   │             ├─ Доплата за выезд
   │             └─ Способы оплаты
   │                │
   │                └─→ 3. Профиль мастера создан ✅
   │                      │
   │                      └─→ 4. Доступны функции мастера:
   │                            ├─ Создание постов
   │                            ├─ Управление записями
   │                            ├─ Настройка профиля мастера
   │                            ├─ Просмотр статистики
   │                            └─ Автопредложения (премиум)
```

### 1.5 Флоу мастера: Управление записями

```
1. Записи (4-я вкладка)
   │
   ├─ Переключатель: [Клиент] [Мастер] [Все записи]
   │  │
   │  └─→ Выбрать "Мастер"
   │        │
   │        ├─ Неподтвержденные (5)
   │        │  │
   │        │  └─→ 2. Новая запись от клиента
   │        │        │
   │        │        └─→ 3. Действия:
   │        │              ├─ ✅ Подтвердить
   │        │              ├─ ❌ Отклонить
   │        │              └─ 🔄 Предложить другое время
   │        │                   │
   │        │                   └─→ 4. Запись подтверждена
   │        │                         ├─ → Подтвержденные
   │        │                         ├─ Push клиенту
   │        │                         └─ Чат доступен
   │        │
   │        ├─ Подтвержденные (12)
   │        │  │
   │        │  └─→ 5. В день визита
   │        │        │
   │        │        └─→ 6. Завершить запись
   │        │              ├─ ОБЯЗАТЕЛЬНО: Оставить отзыв клиенту
   │        │              │  (Без отзыва нельзя завершить!)
   │        │              │
   │        │              └─→ 7. Запись завершена
   │        │                    └─ → История
   │        │
   │        └─ История (156)
   │           │
   │           └─→ 8. Предложить запись клиенту
   │                 ├─ Выбрать клиента из истории
   │                 ├─ Выбрать услугу
   │                 ├─ Предложить дату/время
   │                 │
   │                 └─→ 9. Предложение отправлено
   │                       └─ Клиент получает сообщение в чате
```

### 1.6 Флоу мастера: Создание поста

```
1. Профиль мастера (свой)
   │
   ├─ Кнопка "+" (создать пост)
   │  │
   │  └─→ 2. Создание поста
   │        │
   │        ├─ Выбор медиа (фото/видео, до 10 шт)
   │        │
   │        ├─ Описание (текст, до 2000 символов)
   │        │
   │        ├─ Упоминания (@username)
   │        │  └─ Выбор из клиентов, которым оказывалась услуга
   │        │
   │        ├─ Хештеги (#маникюр #красота)
   │        │
   │        └─ Отметить услугу (из своего каталога)
   │           │
   │           └─→ 3. [Опубликовать]
   │                 │
   │                 └─→ 4. Пост опубликован ✅
   │                       ├─ Появляется в Feed подписчиков
   │                       ├─ Уведомления отмеченным пользователям
   │                       └─ Доступен для лайков/комментариев
```

### 1.7 Флоу пользователя: Друзья и подписки

```
1. Профиль другого пользователя
   │
   ├─→ Добавить в друзья 🤝
   │   │
   │   └─→ 2. Запрос отправлен
   │         ├─ Уведомление получателю
   │         │
   │         └─→ 3. Получатель подтверждает
   │               │
   │               └─→ 4. Стали друзьями ✅
   │                     ├─ Уведомление отправителю
   │                     ├─ Появляются в списках друзей
   │                     └─ Доступна переписка
   │
   └─→ Подписаться 📌 (если мастер или любой пользователь)
       │
       └─→ 2. Подписка оформлена мгновенно
             ├─ Уведомление мастеру
             ├─ Посты в ленте Feed
             └─ Можно отписаться

Фильтр "Друзья" в Feed:
    └─→ Показывает только посты друзей
```

### 1.8 Флоу: Обязательный отзыв

```
Сценарий 1: Клиент завершил услугу, но не оставил отзыв
──────────────────────────────────────────────────────────

1. Клиент пытается создать новую запись
   │
   └─→ 2. Система проверяет: есть ли завершенные записи без отзыва?
         │
         ├─ НЕТ → Переход к созданию записи
         │
         └─ ДА → 3. Блокировка с диалогом
                   │
                   ┌─────────────────────────────────────────┐
                   │ ⚠️ Вы не можете создать новую запись,   │
                   │    пока не оставили отзыв на            │
                   │    завершенную запись                   │
                   │                                         │
                   │         [Оставить отзыв]                │
                   └─────────────────────────────────────────┘
                   │
                   └─→ 4. Клик "Оставить отзыв"
                         │
                         ├─ Если 1 запись → сразу форма отзыва
                         └─ Если >1 записей → список всех без отзывов
                             │
                             └─→ 5. Выбор записи → Форма отзыва
                                   ├─ Рейтинг (1-5 звезд)
                                   ├─ Текст отзыва
                                   └─ [Отправить]
                                      │
                                      └─→ 6. Отзыв оставлен ✅
                                            └─ Теперь можно создавать новую запись

Сценарий 2: Мастер завершает запись
────────────────────────────────────

1. Мастер нажимает "Завершить запись"
   │
   └─→ 2. Обязательная форма отзыва
         ├─ Рейтинг клиента (1-5 звезд)
         ├─ Текст отзыва (опционально, но рекомендуется)
         └─ [Завершить с отзывом]
            │
            ├─ Кнопка "Завершить" неактивна без рейтинга
            │
            └─→ 3. Запись завершена + отзыв отправлен ✅
                  ├─ Статус → Completed
                  ├─ → История
                  └─ Клиент получает уведомление о завершении
```

## 2. Информационная архитектура

### 2.1 Структура приложения (v2.0)

```
Service Platform v2.0
│
├─── 🏠 Feed (Главная)
│    ├─ Лента контента (3 столбца)
│    ├─ Фильтры (геолокация, категории, друзья, подписки)
│    ├─ Поиск
│    └─ → Полноэкранный просмотр поста
│         ├─ Лайк / Комментарий / Поделиться / Избранное
│         ├─ → Профиль мастера
│         └─ → Записаться
│
├─── 🔍 Поиск
│    ├─ Строка поиска
│    ├─ Многоуровневый каталог
│    ├─ Фильтры
│    ├─ Результаты (табы: Мастера, Услуги, Посты, Пользователи)
│    └─ → Избранное (отдельная страница)
│         ├─ Таб: Мастера
│         └─ Таб: Посты
│
├─── 💬 Чаты
│    ├─ Строка поиска чатов
│    ├─ Закрепленные чаты
│    ├─ Список чатов (по последнему сообщению)
│    └─ → Чат с пользователем
│         ├─ Текст / Фото / Видео / Голос / Локация
│         ├─ Sharing (профиль мастера, пост)
│         └─ Предложение записи (от мастера)
│
├─── 📅 Записи
│    ├─ Переключатель: Клиент / Мастер / Все
│    │
│    ├─ Режим "Клиент":
│    │  ├─ Таб: Предстоящие (Pending, Confirmed, In Progress)
│    │  ├─ Таб: История (Completed, Cancelled)
│    │  └─ Кнопка: Новая запись (блокируется если есть записи без отзывов)
│    │
│    └─ Режим "Мастер":
│       ├─ Таб: Неподтвержденные
│       │  └─ Действия: Подтвердить / Отклонить / Предложить другое время
│       ├─ Таб: Подтвержденные
│       │  └─ Действие: Завершить (с обязательным отзывом)
│       └─ Таб: История
│          └─ Действие: Предложить запись клиенту
│
└─── 👤 Профиль
     ├─ Иконка: 🔔 Уведомления (badge с счетчиком)
     ├─ Иконка: ⚙️ Настройки
     │
     ├─ Свой профиль:
     │  ├─ Фото + Имя
     │  ├─ Статистика (для мастера)
     │  ├─ Кнопка: "Стать мастером" (если клиент)
     │  ├─ Кнопка: "Настроить профиль мастера" (если мастер)
     │  ├─ Друзья / Подписки / Подписчики
     │  ├─ Посты (если есть)
     │  │
     │  └─ Для мастера:
     │     ├─ Вкладки: Посты | Портфолио | Услуги | Отзывы
     │     └─ Кнопка: "+" (создать пост)
     │
     └─ Профиль другого пользователя:
        ├─ Фото + Имя
        ├─ Рейтинг и отзывы
        ├─ Кнопки: Добавить в друзья / Подписаться / Написать
        ├─ Друзья / Подписчики (без подписок)
        ├─ Посты
        │
        └─ Для мастера:
           ├─ Вкладки: Посты | Портфолио | Услуги | Отзывы
           └─ Кнопка: "Записаться"
```

### 2.2 Навигационная иерархия

```
Уровень 1: Bottom Navigation (5 вкладок)
├─ Feed
├─ Поиск
├─ Чаты
├─ Записи
└─ Профиль

Уровень 2: Главные страницы
├─ Feed → Полноэкранный просмотр → Профиль мастера → Запись
├─ Поиск → Результаты → Профиль мастера → Запись
├─ Чаты → Чат → Профиль пользователя
├─ Записи → Детали записи → Чат с мастером
└─ Профиль → Настройки / Уведомления / Профиль мастера

Уровень 3: Дополнительные страницы
├─ Избранное (Мастера / Посты)
├─ Уведомления (список)
├─ Друзья (список друзей)
├─ Создание поста (для мастера)
├─ Настройка профиля мастера
└─ Стать мастером (flow 5 шагов)
```

## 4. Детальное описание экранов

### 4.1 Feed (Главная лента)

**Назначение:** Главная страница приложения с лентой контента мастеров

**Layout:**
```
┌─────────────────────────────────────────┐
│ [≡]  SERVICE                       [🔍] │ ← App Bar
├─────────────────────────────────────────┤
│ 📍 20 км | 📂 Все | 👥 Друзья | 📌 Подп.│ ← Фильтры (чипы)
├─────────────────────────────────────────┤
│                                         │
```
